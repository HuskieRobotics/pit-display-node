# Pit Display App

Description: The Robotics Pit Display is an application that is used during FIRST Robotics competitions. It lists upcoming matches, match history, ranking information, as well as a live Twitch display on the first page. The second page is for monitoring the team’s robot functionalities, with the application connecting to the robot’s local server to gather information such as the temperature of the motors as well as the voltage input of different components of the robot. A checklist is included to help your team organize all checks on the robot while it is in the pit.

Based on Ranger Robotics' Flutter app.
